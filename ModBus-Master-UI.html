MBModbus RTU Master

v1.3.0 — Web Serial Client

A browser-based Modbus RTU master using the Web Serial API. Supports FC01–FC06, FC15, FC16 with device profiles, per-register data type interpretation, minus offset addressing, and raw frame inspection.
